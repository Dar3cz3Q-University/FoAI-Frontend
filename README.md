## Installation

1. Clone repository
    ``` shell
    git clone git@github.com:Dar3cz3Q-University/FoAI-Frontend.git
    ```
2. Install dependencies
    ``` shell
    npm i
    ```
3. Copy and fill environment file in src directory
    ``` shell
    cp .env.dist .env
    ```
4. Run application
    ``` shell
    npm run dev
    ```
